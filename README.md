# Hospital Data Analysis (SQL)

## Overview
This project analyzes a hospital dataset using SQL to identify trends in patient data, hospital operations, and cost distribution.

## Dataset
- Total Records: 7,400+

## Tools Used
- PostgreSQL
- SQL

## Key Analysis
- Average patient stay duration
- Department-wise performance
- Monthly expenses trends
- City-wise patient distribution

## Key Insights
- Identified high-cost departments
- Found peak admission periods
- Highlighted areas for cost optimization
